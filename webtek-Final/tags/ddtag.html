ddtag

< dd>

The < dd> tag represents a value in a description list (dl). These tag are usually accompanied by a name or term (dt) for which they provide a description. This type of lists can be used for different purposes like metadata topics and values, questions and answers, glossaries, vocabulary lists, et.

Example

< !DOCTYPE html>

< html>

< head>

< title>ddtag< /title>

< /head>

< body>

< style>

dt {

display: block;

}

< /style>

< /head>

< body>

< p>Amazing Healing Plants< /p>

< dl>

< dt>Acacia< /dt>

< dd>Cough, diabetes, excessive urination, impotency, and throat infection< /dd>

< dt>Carambola< /dt>

< dd>Prevents and cures acurvy and cools the system< /dd>

< /dl>

< p>Try and see the effects.< /p>

< /body>

< /html>
