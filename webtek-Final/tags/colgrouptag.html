< colgroup>

The < colgroup> tag represents columns in table. It also group one or more tables for formatting.

Example:

< !DOCTYPE >

< html>

< head>

< title>colgroup tag < / title>

< table>

< colgroup>

< col span="2" style="background-color:skyblue">

< tr>

< td>Introduction to E-commerce< /td>

< td>11:00-12:00 TF< /td>

< /tr>

< tr>

< td>Web Tech < /td>

< td>3:00-4:00 TF< /td>

< /tr>

< tr>

< td>Physics< /td>

< td>10:00- 11:00 MWF< /td>

< /tr>

< /table>

< /body>

< /html>

Output

Introduction to E-commerce 11:00-12:00 TF
Web Tech 3:00-4:00 TF
Physics 10:00- 11:00 MWF
