< a> tag

The < a> tag represents a hyperlink. An important attribute for the < a> element is the "href" which indicate the link's destination.

Example:

The first example, we created a link to a sample page using the < a> element with the href attribute to link it to sample.html

< p>Click on the link to display our sample page: < a href="sample.html">click me< /a>< /p>
