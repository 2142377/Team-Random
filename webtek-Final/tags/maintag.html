main tag

< main>

This < main> tags a container for the central topics of a document or section. This central content may be interpreted as all that's left after you remove ads, headers (header), footers (footer), navigational sections (nav), related links and other such secondary elements.

Example

< !DOCTYPE html>

< html>

< head>

< title>maintag< /title>

< /head>

< body>

< main>

< h1>HTML< /h1>

< p>Stand for Hypertext Mark-up Language< /p>

< article>

< h1>CSS

< p>Stands for Cascading Style Sheet< /p>

< /article>

< /main>

< /body>

< /html>
