< li>

The < li> tag represents list of items, being it ordered in an < ol > tag or unordered < ul>.

Example

< !DOCTYPE html>

< html>

< head>

< title>litag< /title>

< /head>

< body>

< p>JD Restaurant Menu:< /p>

< ol>

< li>JD Rice< /li>

< li>Beef Ampalaya< /li>

< li>Fried Chicken< / li>

< li>Fried chicken< /li>

< h3>Drinks:< /h3>

< li>Brewed Coffee< / li>

< li>Brewed Coffe with milk< /li>

< li>Iced Tea< /li>

< li>Matcha(hot/cold)< /li>

< /ol>

< p>First Customer Order:< /p>

< ul>

< li>Fried Chicken< /li>

< li>Matcha hot< /li>

< /ul>

< /body>

< /html>
