< table>

The < table> tag represents a tabular form in a table

The < table> tag act on these elements:

  • < caption>
  • < colgroup>
  • < col>
  • < thbody>
  • < thead>
  • < tfoot>
  • < tr>
  • < < td>
  • < th>

Example:

< !DOCTYPE >

< html>

< head>

< title>table tag < / title>

< table>

< tr>

< td>Introduction to E-commerce< /td>

< td>11:00-12:00 TF< /td>

< /tr>

< tr>

< td>Web Tech < /td>

< td>3:00-4:00 TF< /td>

< /tr>

< tr>

< td>Physics< /td>

< td>10:00- 11:00 MWF< /td>

< /tr>

< /table>

< /body>

< /html>

Output

Introduction to E-commerce 11:00-12:00 TF
Web Tech 3:00-4:00 TF
Physics 10:00- 11:00 MWF
