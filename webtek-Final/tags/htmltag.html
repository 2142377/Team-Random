html

The documents in an HTML file is usually starts with a < html > tags

which tells the browser that the HTML document is processed. It is also the parent of all

tags in an HTML files, except for the < !DOCTYPE > .Other elements is the descendants of < html > tag.

Example

< !DOCTYPE >

< html>

< head>

< title>WebTek < / title>

< /head>

< body>

< h1>WEB SYSTEMS AND TECHNOLOGIES< / h1>

< p>Subject taken for you to graduate < /p>

< / body>

< / html>
