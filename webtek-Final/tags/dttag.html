< dt>

The < dt> tag defines a term/name in a description list.The < dt> tag is used in conjunction with < dl> and < dd>.

Example

< !DOCTYPE html>

< html>

< head>

< title>dttag< /title>

< /head>

< body>

< style>

dt {

display: block;

}

< /style>

< /head>

< body>

< p>Amazing Healing Plants< /p>

< dl>

< dt>Coral Plant< /dt>

< dd>Used for cleansing the bowels< /dd>

< dt>Musk Mallow< /dt>

< dd>Expels gas from the stomach and intestines< /dd>

< /dl>

< p>Try and see the effect.< /p>

< /body>

< /html>
