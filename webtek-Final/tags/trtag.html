< trtag>

The < tr> represents a row in a table.

Example:

< !DOCTYPE >

< html>

< head>

< title>tr tag < / title>

< table>

< tr>

< th>Name< /th>

< th>Age< /th>

< /tr>

< tr>

< td>Jeann< /td>

< td>Mich< /td>

< td>Jones< /td>

< td>Jomar< /td>

< td>Jonvic< /td>

< td>22< /td>

< td>20< /td>

< td>17< /td>

< td>15< /td>

< td>10< /td>

< /tr>

< /table>

Output

Name Age
Jeann Mich Jones Jomar Jonvic
22 20 17 15 10
