< tbody>

The < tbody> tag specifies which content of data is in the < table> tag

Example:

< !DOCTYPE >

< html>

< head>

< title>tbody tag < / title>

< table>

< thead>

< tr>

< th>Subjects< /th>

< th>Time< /th>

< th>Day< /th>

< th>Instructor< /th>

< /tr>

< /thead>

< tbody>

< td>Introduction to E-commerce< /td>

< td>11:00-12:00< /td>

< td> TF< /td>

< td> Sir Domantay< /td>

< /tr>

< tr>

< td>Web Tech < /td>

< td>3:00-4:00 < /td>

< td>TF < /td>

< td>Sir Montes < /td>

< /tr>

< tr>

< td>Physics< /td>

< td>10:00- 11:00 < /td>

< td>MWF< /td>

< td>Sir Antonio< /td>

< /tr>

< /tbody>

< /table>

< /body>

< /html>

Output

Subjects Time Day Instructor
Introduction to E-commerce 11:00-12:00 TF Sir Domantay
Web Tech 3:00-4:00 TF Sir Montes
Physics 10:00- 11:00 MWF Sir Antonio
