< form>

The < form> element inserts form that can collect input from the user

The < input> element

Type Description
< input type="submit"> This type defines one -line text input field
< input type="text"> This type is submit button
< input type="radio"> This type defines one a radio button

Example:

< !DOCTYPE >

< html>

< head>

< title>form tag < / title>

< form action="/action_page.php">

< p> First name:< br>

< p> < input type="text" name="firstname">

< br>

Last name:

< input type="text" name="lastname">
< br>

< input type="submit" value="Submit">

< /form>

Output

First name:

Last name:
