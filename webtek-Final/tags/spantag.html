< span>

The < span> tag for grouping and also applying styles that inline elements, color a text or provide a no visual change

Example

< !DOCTYPE >

< html>

< head>

< title>span tag < / title>

< p>I love < span style="color:green;font-weight:bold">green< /span> color at the same time I love < span style="color:skyblue;font-weight:bold">skyblue< /span> color < /p>

< /body>

< /html>

Output

I love green color at the same time I love skyblue color
