< small>

The < small> tag represents text that it contains as side comments

Example:

< h2>LED flashlight< /h2>

< h3>Product description< /h3>

< ol>

< li>High-intensity ultra-white light< /li>

< li>More than 200 hours of lighting by using high quality batteries < small>(batteries are not included)< /small>.

< li>Suitable for camping, searching, home, engineering, etc.< /li> < /ol>

Output

LED flashlight

Product description

  1. High-intensity ultra-white light
  2. More than 200 hours of lighting by using high quality batteries (batteries are not included).
  3. Suitable for camping, searching, home, engineering, etc.
