< tfoot>

The < tfoot> tag represents the footer section of a < table> tag

Example:

In this example add a color using a style element to identify where is < tfoot> tag < tfoot>

< !DOCTYPE >

< html>

< head>

< title>tfoot tag < / title>

< style>

tbody {color:green;}

tfoot {color:blue;}

< /style>

< table>

< thead>

< tr>

< th>Subjects< /th>

< th>Time< /th>

< th>Day< /th>

< th>Instructor< /th>

< /tr>

< /thead>

< tbody>

< td>Introduction to E-commerce Lab.< /td>

< td>11:00-12:00< /td>

< td> TF< /td>

< td> Sir Domantay< /td>

< /tr>

< tfoot>

< td>Introduction to E-commerce Lab< /td>

< td>11:00-12:30< /td>

< td> MTh< /td>

< td> Sir Gab< /td>

< /tr>

< tfoot>

< /table>

< /body>

< /html>
