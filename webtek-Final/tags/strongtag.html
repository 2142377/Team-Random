< strong>

The < strong> tag gives emphasis or importance to the text it contains.

Example

< html>

< head>

< /head>

< body>

< li>< strong>Emotional abuse, past physical, sexual< /strong>< /li>

< li>Personal conflict with family members or friends< /li>

< li>Genetics /family history< /li>

< /ul>

< / body>

< / html>
