stag

< s>

The < s> tag specifies text that is no longer correct, accurate or relevant.

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

Travel destinations:

< p>< s>Bengaongao Cave, Tublay< /s>< /p>

< p>Mt. Pulag, Kabayan< /p>

< p>Bayokbok Falls, Tublay< /p>

< p>< s>Mines View, Baguio City< /s>< /p>

< /body>

< /html>

Output

Travel destinations:

Bengaongao Cave, Tublay

Mt. Pulag, Kabayan

Bayokbok Falls, Tublay

Mines View, Baguio City
