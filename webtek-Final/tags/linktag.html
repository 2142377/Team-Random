< link>

The < link> tag provides relational information about the document. A link element must have a attribute which is mandatory.

The most link attributes are the:

< alt>

-This attribute defines an alternative text of an image if it cannot be dispalyed. It is not a mandatory where you need to add just in case.

Example

< !DOCTYPE >

< html>

< head>

< title>Just Smile < / title>

< /head>

< body>

< img src="smiley.gif" alt="Smiley face" width="50" height="50">

< / body>

< / html>

< href>

-The href attribute also defines the targeted URL address from an HTML link making a word clickable and goes to the other web page.

id

This attribute create an HTML link for the user to specify part in the website and it mostly used in styling that can be bookmark on the web page.

Example
