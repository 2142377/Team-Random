< style>

The < style > tag contains with CSS style declarations in which in this case the

style information will be embedded in the document itself, between the element's tags. The style element

can only be declared in the < head> or in < body> tag.

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< style> * { color: red;

background-color: red;

border-color: red; } < /style>

< / body>

< / html>
