emtag

< em>

The < em> tag gives emphasis or importance to the text it contains.

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< em>This example will give emphasis on the one that is

marked with the < em > tag < /em>

< /body>

< /html>

Output

This is Web System Technologies subject
