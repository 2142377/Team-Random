< thead>

The < thead> this tag represents the header section of a < table> tag

Example:

< !DOCTYPE >

< html>

< head>

< title>thead tag < / title>

< table>

< thead>

< tr>

< th>Subjects< /th>

< th>Time< /th>

< th>Day< /th>

< th>Instructor< /th>

< /tr>

< /thead>

< tbody>

< td>Introduction to E-commerce< /td>

< td>11:00-12:00< /td>

< td> TF< /td>

< td> Sir Domantay< /td>

< /tr>

< /table>

< /body>

< /html>

Output

Subjects Time Day Instructor
Introduction to E-commerce 11:00-12:00 TF Sir Domantay
