< base>

The < base> tag is used in defining the base of the URL or URI of all the relative links in the HTML document.Once the base URL is defined all relative links it can add a base URL as a prefix. This tag is defined inside the < head > element. It can only have aone base in the HTML document.

The mostly used attributes in the base element:

< a href>

- Which defines the base of the URL

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< p > URL for w3school: < a href="https://www.w3schools.com">W3Schools < /p >< /a>

< p>a tag URL: < a href="tag_a.asp">The a tag< /a>< /p>

< / body>

< / html>

< target>

-Define where the targeted URL has to opened.

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< p > Open link to a new window: < a href="https://www.w3schools.com" target="_blank">Visit W3Schools!< /a> < /a>

< p>a tag URL: < a href="tag_a.asp">The a tag< /a>< /p>

< / body>

< / html>
