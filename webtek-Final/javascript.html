Ajax

is a programming routine with regards to building more complicated, dynamic site pages utilizing a technology known as XMLHttpRequest.

AJAX enables you to do is simply refresh portions of the DOM of a HTML site page as opposed to reloading the whole page. AJAX additionally gives you a chance to work asynchronously, which means your code keeps on running while that piece of your page is attempting to reload.

With intelligent sites and current web principles, AJAX is progressively being exchanged by functions inside JavaScript structures and the official Fetch API Standard.
