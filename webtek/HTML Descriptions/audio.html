< audio> element

The < audio> elements allows us to embed audio resource into a web page.

There are two ways to load a audio resource:

  1. Using the src attribute which is the direct method.

  2. Using the < source> element which leans more on compatibility among browsers.

Example:

In the first example we will be using the src attribute to insert a audio. We'll provide a fallback content so that
users with browsers that is incompatible with the < audio> element will be informed about the problem and
have information on how to proceed. We also specified the size of the audio using the width and height attribute.

< audio src="audio/sample.mp3" width="384" height="288" controls>
< p>
Your browser doesn't support audio playback natively or doesn't have the
required Ogg codecs. You may want to upgrade to a modern browser with more
features, in order to view the audio file.
< /p>
< /audio>

Output:

The second example is using the < source> element. Using the < source> element will have more compatibility with all types of browsers

< audio width="384" height="288" controls>

            < source src="audio/sample.mp3" type="audio/mp4">

            < source src="audio/sample.webm" type="audio/webm">

            < source src="audio/sample.ogg" type="audio/ogg">
< /audio>
