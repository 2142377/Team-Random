< img> element

The < img> element represents an image, the image is not inserted but is embedded through an external resource.

The < img> element has two main attributes: src and alt.
The src attritbute specifies the URL of the image and alt attribute specifies an alternate text for an image.

Example:

In this example we will try to get an image using the < img> element using a URL

< img src="https://s7d2.scene7.com/is/image/PetSmart/5251722?$sclp-prd-main_large$" alt="fish">
