< br> element

The < br> element inserts a line break that makes the text go to the next line.

Example:

In this example, we will not use the < br> element.
Even if this haiku example is formatted it will not show the same result.

< p>
A person to love
A person to have always
A person who cares
< /p>

Output:

A person to love A person to have always A person who cares

If we add the < br> element element at the end of each sentence,
then each sentence will go to the next line

< p>
A person to love< br>
A person to have always< br>
A person who cares< br>
< /p>

Output:

A person to love
A person to have always
A person who cares
