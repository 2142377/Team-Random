< strong> element

The < strong> element gives emphasis or importance to the text it contains.

Example:

This example will give emphasis on the one that is marked with the < strong> element.

< h2>In case of fire< /h2> < ul> < li>< strong>Leave fire area and close doors.< /strong>< /li> < li>Activate fire alarm pull stations.< /li> < li>Attemp to extinguish fire only if you can do it safely.< /li> < li>Report any information about fire to Police or Fire Department.< /li> < /ul>

  • Leave fire area and close doors.
  • Activate fire alarm pull stations.
  • Attemp to extinguish fire only if you can do it safely.
  • Report any information about fire to Police or Fire Department.
