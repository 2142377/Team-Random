< video> element

The < video> elements allows us to embed video resource into a web page.

There are two ways to load a video resource:

  1. Using the src attribute which is the direct method.

  2. Using the < source> element which leans more on compatibility among browsers.

Example:

In the first example we will be using the src attribute to insert a video. We'll provide a fallback content so that
users with browsers that is incompatible with the < video> element will be informed about the problem and
have information on how to proceed. We also specified the size of the video using the width and height attribute.

< video src="video/Twitch.mp4" width="384" height="288" controls>
< p>
Your browser doesn't support video playback natively or doesn't have the
required Ogg codecs. You may want to upgrade to a modern browser with more
features, in order to view the video file.
< /p>
< /video>

Output:

The second example is using the < source> element. Using the < source> element will have more compatibility with all types of browsers

< video width="384" height="288" controls>

            < source src="video/Twitch.mp4" type="video/mp4">

            < source src="video/Twitch.webm" type="video/webm">

            < source src="video/Twitch.ogg" type="video/ogg">
< /video>
