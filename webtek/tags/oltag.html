< ol>

The < ol> This tag represents the ordered list of an item, the < li> is also define the list items.

Example

< !DOCTYPE html>

< html>

< head>

< title>ol< /title>

< /head>

< body>

    < li>Apple< /li>

    < li>Mango< /li>

    < li>Banana

    < /ol>

    < /body>

    < /html>
