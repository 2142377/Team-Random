div tag

< div>

This < div> tag is a generic container with no particular semantic meaning. It's commonly used in web authoring for styling purposes, in conjunction with the style and class attributes.

Example

< !DOCTYPE html>

< html>

< head>

< title>divtag< /title>

< /head>

< body>

< h3>WebTech< /h3>

< div style="background-color:pink">

< p>Web System and Technologies< /p>

< p>This is a subject taken third year first semester.< /p>

< /div>

< /body>

< /html>
