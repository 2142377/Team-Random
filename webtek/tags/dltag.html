dltag

< dl>

The < dl> tag represents a description list that are put together names with the (dt) and values (dd)

Example

< !DOCTYPE html>

< html>

< head>

< title>dltag< /title>

< /head>

< body>

< dl>

< dt>Coral Plant< /dt>

< dd>Used for cleansing the bowels< /dd>

< dt>Musk Mallow< /dt>

< dd>Expels gas from the stomach and intestines< /dd>

< /dl>

< /body>

< /html>
