< head >

The < head > tag gives a command to the body of an html file and will efficiently function.

The head element is also the container of the elements that provides a metadata for the document. The metadata specifies

the a title, a character set, author, and description in the document.

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< h1>WEB TECHNOLOGIES!< / h1>

< p>Subject taken for you to graduate < /p>

< / body>

< / html>
