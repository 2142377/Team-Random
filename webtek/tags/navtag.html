navtag

< nav>

The < nav> tag is define as the set of navigation links, navigating the main section of website.

Example

< !DOCTYPE >

< html>

< head>

< title>navtag < / title>

< /head>

< body>

< nav>

< a href="/html/">HTML< /a>

< a href="/css/">CSS< /a>

< a href="/js/">JavaScript< /a>

< /nav>

< / body>

< / html>
