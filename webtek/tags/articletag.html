articletag

< article>

The < article> tag is a container for blocks of content considered to be an independent from the document or website and therefore it can be viewed, reused and distributed separately.

Example

< !DOCTYPE >

< html>

< head>

< title>Life < / title>

< /head>

< body>

< article>

< h1>Jeremiah 29:11< / h1>

< p> For I know the plans I have for you,” declares the LORD, “plans to prosper you and not to harm you, plans to give you hope and a future. < /p>

< /article>

< / body>

< / html>
