< body>

The < body> tag contains the visible contents of an HTML document

Example

< !DOCTYPE >

< html>

< head>

< title>WebTech < / title>

< /head>

< body>

< h1>Hello!< / h1>

< p>What is your name? < /p>

< p>Girl: My name Jen, how about you? < /p>

< p>Boy: I'm Mic, nice meeting you < /p>

< / body>

< / html>
