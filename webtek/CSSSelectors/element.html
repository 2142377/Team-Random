CSS element Selector

The element selector selects all elements with the stated element name

Examples:

Style all < p> element

< html>
< head>
< style>
p {
background-color: blue;
}
< /style>
< /head>
< body>
< h1>Welcome to My Homepage
< div class="intro">
< p id="name">My name is Mack.< /p>
< p id="sport">I play Basketball.< /p>
< /div>
< p>I'm an NBA All-Star player.< /p>
< p>I am now retired< /p>
< /body>
< /html>
