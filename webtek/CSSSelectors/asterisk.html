CSS * Selector

CSS * Selector

The * selector selects all elements

Elements inside an element can be selected using the * selector.

Examples:

Style all elements using * selector

< html>
< head>
< style>
* {
background-color: blue;
}
< /style>
< /head>
< body>
< h1>Welcome to My Homepage
< div class="intro">
< p id="name">My name is Mack.< /p>
< p id="sport">I play Basketball.< /p>
< /div>
< p>I'm an NBA All-Star player.< /p> < /body> < /html>

Style all elements inside < div>

< html>
< head>
< style>
div * {
background-color: blue;
}
< /style>
< /head>
< body>
< h1>Welcome to My Homepage
< div class="intro">
< p id="name">My name is Mack.< /p>
< p id="sport">I play Basketball.< /p>
< /div>
< p>I'm an NBA All-Star player.< /p>
< /body>
< /html>
