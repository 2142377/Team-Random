CSS element element Selector

The element element selector selects and styles elements inside elements.

Examples:

Style all < p> elements that is inside < div> elements

< html>
< head>
< style>
div, h1 {
background-color: blue;
}
< /style>
< /head>
< body>
< h1>Welcome to My Homepage
< div class="intro">
< h2>My name is Mack.< /h2>
< p>I play Basketball.< /p>
< /div>
< p> I don't like playing golf.< /p> < p> I also don't like playing bowling.< /p> < div>
< p>I'm an NBA All-Star player.< /p>
< p>I am now retired< /p>
< /div>
< /body>
< /html>
