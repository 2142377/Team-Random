.class seletor

The .class selector selects elements with a particular class attribute. To determine the elements with a particular class,
write a period (.) character, then the name of the class.

Examples:

Style all elements with the class="kitchen":

< html>
< head>
< style>
.food {
background: blue;
}
< /style>
< /head>
< body>
< p>My name is Donald.< /p>
< p class="food">I like eating Adobo.< /p>
< p>My name is Dolly.< /p>
< p class="food">I also like eating Adobo.< /p>
< /body>
< /html>

Styles all < h1> elements with class="play"

< html>
< head>
< style>
h1.food {
background-color: blue;
}
< /style>
< /head>
< body>
< h1 class="food">Food Lovers< /h1> < p>My name is Donald.< /p>
< p>I like eating Adobo.< /p>
< p>My name is Dolly.< /p>
< p>I also like eating Adobo.< /p>
< /body>
< /html>
