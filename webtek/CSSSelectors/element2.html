CSS element,element Selector

The element,element selector styles various elements with the similar style, each element name is separated by a comma

Examples:

Style all < div> and < h1> elements

< html>
< head>
< style>
div, h1 {
background-color: blue;
}
< /style>
< /head>
< body>
< h1>Welcome to My Homepage
< div class="intro">
< p id="name">My name is Mack.< /p>
< p id="sport">I play Basketball.< /p>
< /div>
< p>I'm an NBA All-Star player.< /p>
< p>I am now retired< /p>
< /body>
< /html>
